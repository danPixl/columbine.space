Two decades after the domain js.org was registered for the first time, it finally serves its obvious purpose - the JavaScript community.

Right from the beginning opinions about JavaScript varied - some people love the language, others hate it. But there should be no doubt about the awesomeness of the community that made JavaScript one of the most popular programming languages.
In alignment to the free-and-open mindset of the community, JS.ORG made subdomains available for developers and their projects for free. Since this first step we also focused on tracking JavaScript trends and spreading news about it.

CONTRIBUTIONS ARE WELCOME: If you have the passion to start a collaborative project, or if you want to be a member of and contribute to JS.ORG, feel free to contact me.

  • dns

    You might have a site on something like Google Sites or GitHub Pages. Let's give your site a sleek name, like dns.columbine.space. We'd like to point out that this isn't a host -- you'll still be using something like Google Sites to be building the actual site. So instead of columbinespace.github.io/services/DNS/, it'll be something like dns.columbine.space.

  • news

    Ready for some number crunching? STATS.JS.ORG fetches statistical parameters for the most important JavaScript projects on GitHub and stuffs them in a giant table with 10k rows. This project aims to make trends of JavaScript visible.

  • chat

    You have something to share with the community or want to read about the stuff other JavaScript enthusiasts do? NEWS.JS.ORG provides a way to easily spread, consume and discuss JavaScript projects.

  • WASM

    WebAssembly - or wasm for short - is a new binary syntax for low-level code on the web platform. WASM.NEWS is a branch of the JS.ORG initiative and acts as a news aggregator on this upcoming topic.
